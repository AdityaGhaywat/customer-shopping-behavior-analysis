# Customer Shopping Behavior Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior data using SQL, Python (Pandas), Excel, MySQL, and Power BI.

The project includes:
- Data cleaning
- Feature engineering
- SQL analysis
- KPI generation
- Dashboard creation
- Business insight extraction

---

## Tools & Technologies Used
- SQL
- Python (Pandas)
- Excel
- Power BI
- MySQL
- Jupyter Notebook

---

## Key Tasks Performed
- Handled missing values
- Cleaned and standardized column names
- Created analytical columns like `age_group` and `purchase_freq_days`
- Connected Python with MySQL
- Performed customer and category-level analysis
- Built an interactive dashboard

---

## Key Insights
- Clothing category generated the highest revenue
- Young Adult customers contributed the highest sales
- Subscription status influenced purchasing behavior

---

## Business Problem
The objective of this project was to analyze customer purchasing behavior and generate insights that can support:
- Inventory planning
- Customer segmentation
- Revenue analysis
- Marketing decisions
